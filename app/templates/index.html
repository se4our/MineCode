Изучите магию командных блоков Minecraft

Ванильный Minecraft скрывает огромный потенциал: десятки команд, механик выполнения и гибкая система тегов. Создавайте мини-игры, автоматизируйте процессы и меняйте облик игры — без единого мода.

Команды и командные блоки активно применяются на серверах, в приключенческих картах и для обучения программированию. Станьте мастером Minecraft-логики!

🕰️ Эволюция команд Minecraft (1.8 – 1.21)

Система команд прошла долгий путь от простых консольных операторов до гибкого языка с поддержкой функций, датапаков и сложной логики. Каждое обновление приносило новые возможности, улучшая инструментарий для создателей карт и серверов.

1.8 – Bountiful Update (2014)

Добавлены команды /fill, /clone и /blockdata, появились первые командные блоки различных типов (импульсный, цепной, повторяющийся). Реализована поддержка JSON-текста в командах /tellraw и /title.

1.9 – Combat Update (2015)

Введены цепные командные блоки, позволяющие выполнять последовательности команд. Улучшена обработка селекторов цели, добавлены сортировка sort=nearest и sort=furthest. Появилась команда /particle.

1.10 – Frostburn Update (2016)

Добавлена команда /teleport для перемещения сущностей с сохранением ориентации. Появились блоки структуры (Structure Blocks) и команда /stopsound.

1.11 – Exploration Update (2016)

Введён наблюдатель (Observer) и связанные с ним механики, включая детектор обновлений блоков. Команда /locate поможет найти структуры.

1.12 – World of Color Update (2017)

Появились функции (functions) — текстовые файлы с последовательностями команд, выполняемые без командных блоков. Добавлена команда /reload для перезагрузки данных без перезапуска сервера. Появились /function и /advancement.

1.13 – Update Aquatic (2018)

Революционный «Flattening» — числовые ID блоков и предметов заменены строковыми, что сильно изменило синтаксис команд. Появились датапаки, позволяющие организовывать функции, достижения, рецепты и таблицы добычи. Команды /execute полностью переписаны с подкомандами as, at, if, unless, что сделало её мощнейшим инструментом.

1.14 – Village & Pillage (2019)

Добавлены команды для работы с торговлей жителей (/data может изменять trades). Расширена гибкость /execute с подкомандой store, позволяющей сохранять результаты в счётчики, блоки или NBT сущностей.

1.15 – Buzzy Bees (2019)

Небольшие улучшения команд: добавлены возможности работы с пчёлами, подкоманда /data получила уточнения. В основном версия сфокусирована на производительности.

1.16 – Nether Update (2020)

Появилась команда /attribute для изменения характеристик сущностей (здоровье, урон и др.). /execute научилась работать в указанном измерении с помощью in. Добавлены новые цели для счётчиков и тегов.

1.17 – Caves & Cliffs (часть 1, 2021)

Команда /item получила подкоманду modify для изменения предметов прямо в инвентаре блока или сущности. Добавлены новые блоки и предметы, доступные через команды.

1.18 – Caves & Cliffs (часть 2, 2021)

Глобальные изменения генерации мира, новые возможности для командной настройки мира. Улучшена команда /placefeature для размещения декораций. Продолжено развитие /item.

1.19 – The Wild Update (2022)

Добавлены команды для работы с Ward-эффектами и Darkness. Расширен синтаксис /execute добавлением условий if biome, if loaded. /place template позволяет размещать структуры из датапаков.

1.20 – Trails & Tales (2023)

Введена команда /damage для нанесения урона с контролируемой причиной и звуком. Команда /ride управляет посадкой сущностей друг на друга без сложного NBT. Добавлена поддержка /return в функциях для возврата значений.

1.21 – Tricky Trials (2024)

Добавлены команды для Trial Chambers: /execute получила новые условия if items для проверки инвентаря. /item улучшена для работы с хранилищами. Расширены возможности кастомизации через макросы в функциях.

С каждой версией командный арсенал Minecraft становится всё более приближенным к полноценному языку программирования, открывая безграничные творческие возможности.
